# Security Policy

This repository is pre-production. Do not use it for real escrowed funds until the payment and escrow launch controls are approved.

Report suspected vulnerabilities privately to the Bittrees Technology maintainers. Include:

- Affected component.
- Reproduction steps.
- Expected impact.
- Suggested mitigation, if known.

Before launch, this project needs review for wallet authentication, buyer/provider identity, signer authority, escrow release/refund/dispute logic, input validation, secrets, deployment configuration, marketplace spam, provider impersonation, and abuse controls.
